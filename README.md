✨ Features : 

👤 User Panel :
Register & Login with JWT Authentication
Browse doctors by speciality
View doctor profiles & availability
Book appointment slots in real-time
Online payment via Razorpay
View & cancel appointments
Profile management 

🩺 Doctor Panel:
Dedicated login & dashboard
View earnings, appointments & patients count
Mark appointments as completed or cancel
Update profile, fees & availability

🛠️ Admin Panel: 
Add & manage doctors
View all appointments with status
Cancel appointments
Dashboard with earnings, doctors, patients & stats
